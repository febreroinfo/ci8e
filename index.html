What is it?

A tool that assists with project layout problems. It is a fork of Pesticide that I found useful in my development. I extended the functionality to be keyboard key press activated with the options to toggle between the standard outline option or the enhanced option with box shadow to show element depth on hover.

Why use it?

Sometimes you need to tell elements apart to identify what's causing the layout issue. This is where this tool comes in and shines.

Time saver

Saves time by limiting the need to click around the browser DevTools to identify issues. The tool outlines every page element and helps tell each piece apart.

Nondestructive

Nondestructive since the script inserts the debugging style into the current page dynamically. Once the project is cleared for Production, the one-line script tag can be removed.

Offline friendly

The script is standalone and therefore can be used offline without needing to be connected online. Perfect for offline development.

Option Controls

Use the following keyboard combination to activate and deactivate the tool.

Focus on page then,  Ctrl + mouseover  or Cmd + mouseover to display element attributes

Focus on page then,  Ctrl + ~ or Cmd + ~ to enable Outline option

Focus on page then,  Ctrl + 1 or Cmd + 1 to enable Outline and Box Shadow option

Focus on page then,  Ctrl + 2 or Cmd + 2 to disable any active option

Alternatively,

Ctrl + Previous Key option Pressed or Cmd + Previous Key option Pressed to disable the active option
